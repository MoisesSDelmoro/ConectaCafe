# Conecta Café #
## Sobre 
---
O projeto **Conecta Café** foi um projeto desenvolvido para a disciplina de C317 (Tópicos Especiais) no [Inatel](https://inatel.br/home/), com o intuito de desenvolvermos um projeto com Front-End, Back-End, persistência de dados e testes autômatizados, em uma plataforma Web e outra Mobile. Seu objetivo é facilitar a comunicação do produtor de café com seus clientes finais sobre a venda de cafés especiais.

## Como utilizar?
- Instale e configure o Flutter, bem como as variáveis de ambiente [Link](https://flutter.dev/docs/get-started/install/).

```bash 
# Clone o repositório em seu diretório
$ git clone https://github.com/MoisesSDelmoro/ConectaCafe
  
# Entre no diretório
$ cd ConectaCafe
  
# Se preferir executá-lo pela linha de comando, sigua:
$ flutter upgrade
$ flutter pub get 
$ flutter run
  
# Para acessar o código fonte:
$ code .
```
